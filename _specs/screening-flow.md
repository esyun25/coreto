# KYC・書類審査フロー仕様
最終更新: 2026-03-03

## 1. 審査フロー
1. 申請者が入会フォームを送信
2. Zapier → Kintoneに登録 → Slack #新規申請 に通知
3. 審査担当（manager以上）がscreening.htmlで確認
4. 書類確認（AI支援: 身分証・資格証のOCR読み取り・チェックリスト自動入力）
5. 書類審査チェックシート記入
6. 面接実施（Zoom or 対面）
7. 審査結果: 合格 / 不合格 / 保留

## 2. 書類チェック項目（RE: 不動産エージェント）
- 氏名・住所・生年月日（身分証と一致）
- 身分証の有効期限
- 宅建士証の有効期限・登録番号（宅建士申請者のみ）
- 顔写真の一致確認
- 反社チェック

## 3. 書類チェック項目（PT: パートナー）
- 個人: 身分証
- 法人: 登記簿謄本 + 代表者身分証
- LINE WORKS登録情報との一致確認

## 4. AI活用ポイント（予定）
書類アップロード時にHaikuで身分証・宅建士証の氏名・有効期限・番号を自動抽出し
チェックリスト項目にプレ入力。HQは確認・承認のみ行う。
約800トークン/件。月50件で月数百円程度。

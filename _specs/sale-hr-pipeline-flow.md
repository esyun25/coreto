# 案件管理フロー（売買・三為・人材紹介）仕様
最終更新: 2026-03-03
賃貸仲介フローは rental-flow.md 参照。売買・三為は sale-sannai-flow.md 参照。

## HR（人材紹介）案件フロー

新規案件登録（type: hr）
  → 面接調整（apply）: 候補者に求人を紹介・複数企業への同時応募管理
  → 内定・入社（contract）: 内定承諾・入社日確定・手数料登録
  → 在籍確認（payment）: 入社後に在籍確認 → 第2回報酬
  → 完了（done）

### 複数応募管理
1案件で複数企業に同時応募できる。各企業のステータスを独立管理:
書類選考 → 面接 → 内定 → 内定承諾 / 不採用 / 辞退

### 在籍確認と第2回報酬
- 在籍確認完了 → 第2回報酬が月次振込に組み込まれる
- 在籍確認未完了で期限超過 → 翌月に延期
- 退職報告あり → early-quit-flow.md の返金フローへ

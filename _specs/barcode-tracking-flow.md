# バーコード・書類追跡 運用フロー仕様
最終更新: 2026-03-03

## 1. 用途
賃貸契約書のレターパック郵送における書類追跡管理。

## 2. 運用フロー
1. HQが書類受取 → barcode-scan.htmlの「書類受取」タブ
2. 案件ID・書類種別・管理会社・送付先を入力
3. バーコードラベル（Code128）印刷 → レターパックに貼付
4. 郵便局に差出し → 追跡番号をシステムに入力
5. Zapier（1日1回）が郵便局APIで配達状況を自動チェック
6. 配達完了時にSlack通知
7. 受取確認 → 案件ステータスを「書類受取完了」に更新

## 3. 書類種別
- 契約書（管理会社→クライアント）
- 署名済み書類（クライアント→管理会社）
- 重要事項説明書
- その他（任意入力）

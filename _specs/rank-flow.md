# ランク昇格・降格フロー仕様
最終更新: 2026-03-03

## 1. 評価タイミング
毎月末日に自動評価。翌月1日から新ランクが適用される。

## 2. 昇格フロー
1. 月末にランク判定バッチ実行
2. 今月ポイント >= 昇格閾値 かつ 連続達成条件を満たす → 昇格確定
3. HQが確認・承認（agent-mgmt.html）
4. Slack #ランク変更 に通知（AG名・旧ランク→新ランク）
5. AGのhub.htmlにバナーで昇格通知
6. 祝福メールを送付
7. 翌月1日から新ランク報酬率を適用

## 3. 降格フロー
1. 今月ポイント < 維持閾値 かつ 猶予期間超過 → 降格確定
2. Slack #ランク変更 に通知
3. AGのhub.htmlにアラートバー表示
4. 降格通知・次月目標ポイントをメールで案内

## 4. ランク保証（Lifetime）
条件を満たしたAGはランク保証が付与される。保証付きAGは降格しない。

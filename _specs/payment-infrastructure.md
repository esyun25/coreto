## 1. 資金フローの全体像

CORETOには3方向の資金フローがある。

```
① エージェント → CORETO    月会費（¥490 or ¥980）
② CORETO → エージェント    報酬・コミッション支払い
② CORETO → パートナー      報酬・コミッション支払い
② CORETO → メンター        メンターフィー支払い
③ クライアント → CORETO    仲介手数料・紹介料受取
```

---

## 2. サービス別の役割分担

| フロー | 金額帯 | 使用サービス | 理由 |
|---|---|---|---|
| ① 月会費（AG→CORETO） | ¥490〜¥980/月 | **Stripe** | 定期引き落とし、カード決済 |
| ② 報酬支払い〜99万円（CORETO→AG/PT） | 〜¥990,000 | **Kyash法人送金** | 口座情報不要、即時、低コスト |
| ② 報酬支払い100万円以上（CORETO→AG） | ¥1,000,000〜 | **GMOあおぞらネット銀行 振込** | Kyashの上限を超える案件のみ |
| ③ 仲介手数料受取（クライアント→CORETO） | 数十万〜数百万 | **GMOあおぞらネット銀行 バーチャル口座** | 請求書発行+銀行振込が業界標準 |
| 即時払い申請（AG申請時） | 任意 | **Kyash or GMO**（金額で分岐） | 15%手数料、2FA必須 |

## 3. GMOあおぞらネット銀行

### 用途①：クライアントからの入金受取（バーチャル口座）

賃貸・売買・人材紹介の仲介手数料をクライアントから受け取る際に使用。

**運用フロー**

```
案件成約
  ↓
HQが請求書発行（coreto-hq-receipt.html）
  ↓
クライアントがCORETOのGMO口座に振込
  ↓
GMO BaaS API webhook → Zapier → Kintone（入金フラグ自動更新）
  ↓
入金確認後、成約処理が進む（coreto-batch-close.html Step1）
```

**特徴**
- バーチャル口座を案件ごとに発行し、入金を自動識別することも可能
- 法人クライアント（不動産会社・建設会社等）は銀行振込が業界標準のため最適
- GMO BaaS APIの申請審査には約2ヶ月かかる（早期申請が必要）
- API連携費用：**無料**（GMOあおぞらネット銀行との契約内）

### 用途②：エージェントへの高額報酬振込（100万円以上）

```
月次締め処理（coreto-batch-close.html）
  ↓
「GMO振込CSVエクスポート」ボタンで振込データ生成
  ↓
GMOあおぞらネット銀行の「総合振込」画面にCSVアップロード
  ↓
一括振込実行（毎月25日）
```

**注意点**
- 100万円以上のコミッションが発生するのはGold・Platinumランクの高額案件のみ
- このケースのみ、エージェントの口座情報（銀行名・口座番号）が必要
- 口座情報はKintoneに暗号化して保存（KYC済み・専任宅建士フラグのあるエージェントのみ）

### 月会費のGMO連携（Stripe連動）

- Bronze/SilverエージェントはStripeで月会費を自動引き落とし
- HQアクセス付与時：Zapier経由でGMOサブスクリプション停止
- HQアクセス解除時：Zapier経由でGMOサブスクリプション再有効化
- Gold・Platinumは月会費¥0のため引き落とし対象外

## 4. Kyash法人送金

### 対象
CORETO → エージェント・パートナー・メンターへの報酬支払いで**99万円以下**の案件

### 特徴・制約

| 項目 | 内容 |
|---|---|
| 1件あたり送金上限 | **100万円**（99万円以下で運用） |
| 口座情報の要否 | **不要**（電話番号またはKyashアカウントで送金） |
| 受取方法 | Kyashアプリ（スマホ）または銀行口座に出金 |
| 送金手数料 | 銀行振込より安価（件数割引あり） |
| 送金速度 | 即時〜翌営業日 |
| API連携 | 可能（Zapier経由でも可） |
| 導入審査 | オンライン申込み、最短2週間でスタート可能 |

### Kyashを使う理由
CORETOのエージェントの大多数（Bronze・Silver）は1案件あたりのコミッションが100万円未満に収まる。口座情報を管理しないことで、情報漏洩リスクを大幅に削減できる。

### 月次振込フロー（Kyash経由）

```
coreto-batch-close.html で月次承認完了
  ↓
Zapier が Kyash API を叩いて一括送金指示
  ↓
エージェント・パートナーのスマホに通知
  ↓
Kyashアプリで受取 or 銀行口座に出金
  ↓
Kintoneに振込完了フラグを書き戻し
```

### 100万円ルーティング分岐

```
月次報酬計算（coreto-payroll.html）
  ├─ 振込額 < 100万円 → Kyash送金（自動）
  └─ 振込額 ≥ 100万円 → GMO振込CSV（手動・HQが実行）
```

## 5. 即時払い（Instant Pay）

### 概要
エージェントが翌月25日の通常振込を待たず、確定した報酬を即時で受け取れる機能。

### 手数料
**申請額の15%**（エージェント負担）

### フロー

```
エージェントが coreto-instant-pay.html で申請
  ├─ 申請金額を入力（最低¥10,000）
  ├─ 2FA認証（SMS）
  └─ Slack #即時払い に通知
      ↓
HQが管理画面（coreto-instant-pay.html HQビュー）で確認
  ├─ 振込額 < 100万円 → Kyashで振込実行
  └─ 振込額 ≥ 100万円 → GMO銀行から手動振込
      ↓
振込完了 → Kintoneに記録・エージェントに通知
```

### 2FA要件
- SMS認証（登録済みの電話番号に送信）
- 認証コードの有効期限：10分
- account-settings.html で電話番号の設定が必要
- 未設定の場合は即時払い申請不可

## 6. Stripe（月会費）

### 用途
エージェントからCORETOへの月会費の自動引き落とし

### 金額

| ランク | 月会費 |
|---|---|
| Bronze | ¥980/月 |
| Silver | ¥490/月 |
| Gold | ¥0 |
| Platinum | ¥0 |
| 特別紹介コード適用者 | ¥0（永久無料） |

### 運用
- 毎月1日のランク判定後に翌月の引き落とし金額が確定
- 月会費は報酬振込時に差し引いて振込（Stripeによる引き落としではなく報酬からの控除方式）
- Stripe顧客IDはKintoneで管理

---

## 7. 口座情報の管理方針

セキュリティリスクを最小化するため、以下の方針を採用。

| 区分 | 方針 |
|---|---|
| 〜99万円のエージェント | **口座情報を取得・管理しない**（Kyash送金） |
| 100万円以上が発生したエージェント | **その時点のみ**口座情報を別途取得（LINE WORKSまたはメールで暗号化送付、Kintoneに保存） |
| 口座情報取得条件 | KYC済み、かつ専任宅建士フラグがある場合に限定 |

この方針により、口座情報を保有するエージェント数をGold・Platinumランクの一部に絞り込み、情報漏洩リスクを最小化する。

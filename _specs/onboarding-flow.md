## 2. エージェント入会フロー

### 前提：招待制（トークンゲート）
入会ページは**面接合格後にHQが発行する招待トークン付きURL**でのみアクセス可能。
書類審査合格の時点ではURLは発行されない。

### 全体フロー

```
① 入会申請フォーム送信
      ↓
② 書類審査（coreto-screening.html）
   担当：manager以上限定
   所要：RE 10〜20分 / HR 15〜30分 / PT 5〜10分
   結果：合格 / 不合格 / 保留
      ↓
  不合格 → 不合格通知メール送信（Zapier + Gmail）→ 終了
  保留   → 追加確認後に再判定
      ↓
③ 面接予約リンク発行（Calendly）
   書類審査合格後、HQがCalendlyリンクをメールで送付（有効期限7日間）
      ↓
④ 面接実施（Zoom / 対面 どちらも可）
   担当：manager以上
      ↓
  不合格 → 不合格通知メール → 終了
      ↓
⑤ 面接合格 → 入会URL（招待トークン）発行・SMSで送付
      ↓
⑥ 入会手続き（全8ステップ）
   coreto-agent-signup.html / coreto-partner-signup.html
      ↓
⑦ HQがeKYC確認（書類の目視確認・DB照合）
      ↓
⑧ IDとパスワードをSMSで送付 → ログイン開始
      ↓
⑨ オンボーディング（coreto-onboarding.html）
```

### 入会申請8ステップ（面接合格後・招待URLアクセス時）

```
STEP 1: 基本情報入力
  ├─ 氏名・フリガナ・生年月日・住所
  ├─ 電話番号・メールアドレス
  ├─ 業種選択（不動産 / 人材）
  ├─ 業界経験・得意エリア・志望動機
  └─ 紹介コード（任意）

STEP 2: 宅建士資格確認（不動産エージェントのみ）
  ├─ 宅建士あり → 専任登録 or 一般登録を選択
  │    専任登録：宅建士証番号・登録都道府県を入力
  │    ※他社での専任宅建士登録は不可（宅建業法）
  └─ 宅建士なし → 一般エージェントとして継続

STEP 3: SMS認証
  └─ 登録電話番号にOTPを送信（Zapier + Twilio）
       有効期限5分・最大5回試行・最大3回再送

STEP 4: 身分証明書アップロード
  ├─ 運転免許証（表裏）
  ├─ パスポート（顔写真ページ）
  └─ マイナンバーカード（表裏）

STEP 5: 口座情報入力（100万円以上の案件が発生した場合のみ必要）
  ├─ **Kyash導入前提のため、入会時点では口座情報を収集しない**
  ├─ 99万円以下の振込はKyashで口座情報不要
  └─ 100万円以上の振込が初めて発生した時点でHQが個別に収集
       → KYC済み・専任宅建士フラグのあるエージェントのみ対象

STEP 6: 決済情報（Stripe）
  ├─ 専任宅建士：月会費¥0・登録費¥0 → カード登録不要
  └─ 一般エージェント：月会費¥980〜・名刺費¥2,000 → Stripe Elements でカード登録

STEP 7: LINE登録
  └─ CORETO公式LINEを友達追加・認証コード受信確認

STEP 8: 確認・送信
  └─ 入力内容確認 → Zapier Webhookで本部に申請データ送信
```

### 送信後のメッセージ

| 種別 | メッセージ |
|---|---|
| 専任宅建士 | 「専任宅建士登録の審査後、本部より連絡いたします。審査完了後にIDとパスワードが発行されます。」 |
| 一般エージェント・HR | 「本部でeKYC確認後、IDとパスワードをSMSでお送りします。」 |

## 3. 書類審査（coreto-screening.html）

申請データ受信後、HQが書類審査チェックシートで審査を実施。

### 審査担当者
**HQのmanager以上限定**（staffは閲覧のみ・承認操作不可）

coreto-screening.html のアクセス制御：
- `hqRole === 'exec'` または `hqRole === 'manager'` のみ審査・承認操作可
- `hqRole === 'staff'` は閲覧・参照のみ

### 審査結果と次のアクション

| 結果 | 次のアクション |
|---|---|
| **合格** | Calendlyリンクをメールで送付（面接予約・有効期限7日間） |
| **不合格** | 不合格通知メールを送付（Zapier + Gmail）→ 終了 |
| **保留** | 追加確認後に再判定 |

| # | チェック項目 | 所要時間 | 即不合格条件 |
|---|---|---|---|
| ① | 基本情報確認（氏名・年齢・住所・連絡先） | 約2分 | 18歳未満・法人希望 |
| ② | 反社チェック（Google検索・Kintone内部DB照合） | 約2分 | 過去除名記録あり |
| ③ | 宅建士資格確認（番号形式・有効期限・証明書画像・国交省DB照合） | 約5分 | 国交省DBと不一致 |
| ④ | 職歴・経験確認（経験年数・実績の具体性・得意エリア） | 約5分 | — |
| ⑤ | 志望動機確認（100文字以上・具体性） | 約3分 | 100文字未満（原則） |
| ⑥ | 紹介者確認（紹介コード・紹介者ランク確認） | 約1分 | — |

**合計所要時間：10〜20分**

### 人材エージェント（HR）の審査項目

| # | チェック項目 | 即不合格条件 |
|---|---|---|
| ① | 基本情報確認 | 18歳未満・法人希望 |
| ② | 反社チェック | 過去除名記録あり |
| ③ | **人材業界経験確認 ★最重要** | 経験1年未満・未経験 |
| ④ | 専門分野確認 | — |
| ⑤ | 志望動機確認（100文字以上） | — |

**合計所要時間：15〜30分**

### パートナー（PT）の審査項目

| # | チェック項目 | 備考 |
|---|---|---|
| ① | 基本情報確認（個人：氏名、法人：法人名＋代表者名） | 18歳未満は即不合格（個人のみ） |
| ② | 反社チェック | 過去除名記録あり → 即不合格 |
| ③ | 法人確認（法人パートナーのみ・国税庁DB照合） | — |
| ④ | 職業・ネットワーク確認 | — |
| ⑤ | 志望動機確認（50文字以上） | — |
| ⑥ | 紹介者確認 | — |

**合計所要時間：5〜10分**

## 4. 面接（書類審査合格者全員必須）

### 対象
書類審査合格者は**全員必須**。合格しても面接をスキップして入会することはできない。

### 方法
Zoom / オンライン または 対面（申請者・HQの状況に応じて選択）

### 予約
- HQがCalendlyリンクをメール送付（書類審査合格後）
- 有効期限：7日間
- 本番：Calendly API + Zapier経由で自動送信

### 担当者
manager以上

### 面接結果と次のアクション

| 結果 | 次のアクション |
|---|---|
| **合格** | 入会招待URL（トークン付き）をSMSで送付 → 入会手続きへ |
| **不合格** | 不合格通知メール送付 → 終了 |

### 入会URLの発行タイミング
**面接合格を確認してから発行**。書類審査合格の時点では発行しない。

## 5. 面接合格後の手続き

### 合格の場合

```
面接合格
  ↓
入会招待URL（?token=xxx）をSMSで送付（coreto-id-manager.html）
  ↓
申請者が入会手続き（全8ステップ）を実施
coreto-agent-signup.html / coreto-partner-signup.html
  ↓
業務委託契約書の電子署名（coreto-contract.html）
  ↓
eKYC完了確認（HQが目視確認・coreto-kyc.html）
  ↓
HQがIDを発行（coreto-id-manager.html）
  ├─ 不動産AG：10001〜19999
  ├─ 人材AG  ：20001〜29999
  └─ パートナー：30001〜39999
  ↓
IDとパスワードをSMSで送付
  ↓
coreto-hub.html（ダッシュボード）へのアクセス開始
  ↓
オンボーディング開始（coreto-onboarding.html）
```

### 不合格の場合
HQが不合格通知メールを送付（Zapier + Gmail）。理由を明記する。

## 7. 月会費・登録費の初期設定

| 種別 | 月会費 | 名刺費 | Stripe登録 |
|---|---|---|---|
| 専任宅建士 | ¥0 | ¥0 | 不要 |
| 一般エージェント（Bronze） | ¥980/月 | ¥2,000 | 必要（入会時にカード登録） |
| HR・人材AG | ランクによる | — | 必要 |
| パートナー | なし | — | 不要 |

---

## 8. 専任宅建士の特別ルール

- 登録後は**他社での専任宅建士登録不可**（宅建業法）
- 他社に転職・就職した場合は必ずCORETOに連絡が必要
- 専任解除はHQ管理職（exec / manager）のみ実行可能（coreto-agent-mgmt.html）
- IT重説対応資格あり → 1件あたり報酬加算

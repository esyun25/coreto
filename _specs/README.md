## 記載ルール

- ステータスは `設計済み・未実装` / `実装中` / `実装済み` / `保留` のいずれかを使う
- 決定事項が変更された場合は上書きせず、変更日と変更理由を追記する
- 実装完了後も削除せず、ステータスを `実装済み` に更新して残す

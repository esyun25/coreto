## 1. ランク制度

### ランク判定
- 毎月1日にKintoneで累計ポイントを自動集計・判定
- 昇降格はSlackで通知

### ランク一覧

| ランク | 必要累計ポイント | 月会費 | 報酬率（不動産） | 報酬率（人材） |
|---|---|---|---|---|
| Bronze | 0〜9pt | ¥980/月 | 65% | 45% |
| Silver | 10〜19pt | ¥490/月 | 67.5% | 47.5% |
| Gold | 20〜29pt | ¥0 | 70% | 50% |
| Platinum | 30pt〜 | ¥0（永久無料） | 75% | 55% |

### ポイント付与ルール

| 条件 | 付与ポイント | 対象 |
|---|---|---|
| 自己案件：手数料売上10万円ごと | 1.0 pt | エージェント |
| 自己案件：案件完了ボーナス | 1.0 pt | エージェント |
| パートナー案件：手数料10万円ごと | 0.5 pt | AG・PT 両方に付与 |
| パートナー案件：完了ボーナス | 0.5 pt | AG・PT 両方に付与 |
| メンターコミッション5,000円ごと | 0.2 pt | メンター |

## 2. メンターコミッション

### 定義
自分が紹介して入会させたエージェント・パートナーが成約するたびに、メンターへコミッションが発生する。

### メンターフィー率

**デフォルト（メンター自身のランクで変わる）**

| メンターのランク | デフォルト率 | 上限 |
|---|---|---|
| Bronze | 5% | 10% |
| Silver | 6% | 10% |
| Gold | 7% | 10% |
| Platinum | 8% | 10% |

計算基準：**総手数料（仲介料）に対するパーセンテージ**

**経営者による個別変更**
- `hqRole === 'exec'`（経営者）のみ、エージェント・パートナーごとのメンターフィー率を自由に変更可能
- 変更範囲：0%〜**10%**（上限10%）
- 変更はKintoneの該当エージェント/パートナーレコードに `mentor_fee_override` フィールドとして保存
- `mentor_fee_override` が設定されている場合、ランクによるデフォルト率より**優先**する
- 変更操作は `coreto-admin-rbac.html` または `coreto-agent-mgmt.html` の経営者専用UIから実施

### メンターフィーの階層
- **1階層のみ**。A→B→Cと紹介があってもCの成約時にAへの報酬は発生しない

### 発生ケース

| 案件タイプ | メンター対象 | 備考 |
|---|---|---|
| 自己持ち込み | **エージェントのメンター** | 発生する |
| パートナー案件 | **パートナーのメンター** | 発生する |
| パートナー案件 | エージェントのメンター | **発生しない** |

### メンターフィーの財源
- 自己持ち込み案件：HQの取り分から支出
- パートナー案件：HQの取り分から支出

## 3. システム利用料（賃貸仲介）

### 金額

| 手数料（仲介料）の金額 | システム利用料 |
|---|---|
| 10万円未満 | ¥10,000 |
| 10万円以上 | ¥20,000 |

### 負担ルール

| 案件タイプ | 負担者 |
|---|---|
| 自己持ち込み | エージェントの報酬から控除（全額） |
| パートナー案件 | エージェントとパートナーで**折半** |

**パートナー案件の折半例**

| 手数料 | システム利用料 | AG負担 | PT負担 |
|---|---|---|---|
| 80,000円 | ¥10,000 | ¥5,000 | ¥5,000 |
| 335,000円 | ¥20,000 | ¥10,000 | ¥10,000 |

## 4. 賃貸仲介 報酬分配まとめ

### 自己持ち込み案件

```
総手数料
├─ エージェント：ランク報酬率（65〜75%）- システム利用料 - 月会費
├─ エージェントのメンター（存在する場合）：手数料 × メンターフィー率（5〜8%）
└─ HQ：残り（メンターフィー・システム利用料含む）
```

**計算例（手数料30万円・Goldエージェント・Bronzeメンターあり）**

| 項目 | 金額 |
|---|---|
| 総手数料 | ¥300,000 |
| エージェント報酬（70%） | ¥210,000 |
| 　源泉徴収（10.21%） | −¥21,441 |
| 　システム利用料 | −¥20,000 |
| 　**エージェント手取り** | **¥168,559** |
| メンター報酬（5%・Bronze） | ¥15,000 |
| 　源泉徴収（10.21%） | −¥1,532 |
| 　**メンター手取り** | **¥13,468** |
| **HQ取り分** | **¥75,000**（25%＋システム利用料¥20,000 − メンター¥15,000） |

---

### パートナー案件

```
総手数料
├─ エージェント：25% - システム利用料の半額 - 月会費
├─ パートナー：25% - システム利用料の半額
├─ パートナーのメンター（存在する場合）：手数料 × メンターフィー率（5〜8%）
└─ HQ：50% − パートナーメンターフィー（メンター存在時）
```

**計算例（手数料30万円・Bronzeエージェント・Bronzeパートナー・パートナーにBronzeメンターあり）**

| 項目 | 金額 |
|---|---|
| 総手数料 | ¥300,000 |
| エージェント報酬（25%） | ¥75,000 |
| 　源泉徴収（10.21%） | −¥7,658 |
| 　システム利用料（折半） | −¥10,000 |
| 　月会費（Bronze） | −¥980 |
| 　**エージェント手取り** | **¥56,362** |
| パートナー報酬（25%） | ¥75,000 |
| 　源泉徴収（10.21%） | −¥7,658 |
| 　システム利用料（折半） | −¥10,000 |
| 　**パートナー手取り** | **¥57,342** |
| パートナーのメンター報酬（5%・Bronze） | ¥15,000 |
| 　源泉徴収（10.21%） | −¥1,532 |
| 　**メンターパートナー手取り** | **¥13,468** |
| **HQ取り分** | **¥135,000**（50% − メンター¥15,000 + システム利用料¥20,000） |

## 5. 月会費の控除タイミング

- 毎月の報酬振込時にその月の月会費を差し引いて振込
- Gold・Platinum は月会費¥0なので控除なし
- 特別紹介コード使用者は月会費永久無料（ランク問わず）
